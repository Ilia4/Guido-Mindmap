# Telegram Mindmap Bot

Первая версия отдельного сервиса для Telegram-бота, который:

- подключается к группе/супергруппе;
- запоминает тему после команды `/sync`;
- хранит подписки локально в `sqlite`;
- раз в день проходит по сохранённым подпискам и запускает заглушку синхронизации;
- готов к следующему шагу: обновление `mindmap` через API.

## Что уже умеет

- `/start` — краткая справка;
- `/sync` — подписывает текущую группу/тему на ежедневный sync;
- `/unsync` — снимает подписку с текущей группы/темы;
- `/status` — показывает текущую подписку;
- ежедневный планировщик запускает sync по всем активным подпискам.

## Что важно для тем

Если команда вызвана в forum topic супергруппы, бот сохраняет:

- `chat_id`
- `message_thread_id`

То есть подписка идёт именно на тему, а не на всю группу целиком.

Если команда вызвана вне темы, бот подписывает весь чат.

## Быстрый запуск

1. Создай `.env` на основе `.env.example`
2. Установи зависимости:

```bash
pip install -r requirements.txt
```

3. Запусти:

```bash
python bot.py
```

## Следующий шаг

Когда дашь токен, можно будет:

- подключить реальный Telegram bot token;
- добавить вызов LLM;
- привязать sync к `mindmap` backend;
- сохранять summary/progress по теме и обновлять проект в `mindmap`.
